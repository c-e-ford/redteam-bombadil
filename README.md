# redteam-bombadil
A sample dataset of adversarial prompts used for a multimodal red teaming study at Appen. The dataset includes responses from four LLMs - GPT 4o, Claude Sonnet 3.5, Pixtral 12B, Qwen VL Plus. Adversarial prompts are both multimodal (text + image) and have a text-only equivalent format.
